ConwayLife – Sprint 1

Introduction

Goal: realizzazione in Java del Game of Life di Conway , limitatamente al requisito R1.

Requirements

R1  Realizzare una versione in Java del gioco Life di Conway, come gioco zero-player.
    Il gioco consiste nell’introdurre una griglia di celle il cui stato
    (viva/morta) evolve secondo le regole di Conway.

R2  L’utente deve poter:
     - specificare la configurazione iniziale della griglia
     - vedere l’evoluzione del gioco
       (vedi “Problema della vista del gioco”)
     - fermare e far ripartire l’evoluzione
     - pulire la configurazione della griglia (a gioco fermo)

Requirement analysis

Dall'analisi dei requisiti vediamo le entità Cella e Griglia, che formalizziamo come di seguito.

ICell

	public interface ICell {
	    void setStatus(boolean v);
	    boolean isAlive();
	    void switchCellState();
	}

IGrid

	public interface IGrid {
	    int getRowsNum();
	    int getColsNum();
	    void setCellValue(int x, int y, boolean state);
	    ICell getCell(int x, int y);
	    boolean getCellValue(int x, int y);
	    void reset();
	}

Dai requisiti R1 leggiamo che abbiamo bisogno di incapsulare le regole del gioco in un'entità. A questo scopo formalizziamo Life.

LifeInterface

	public interface LifeInterface {
	    void nextGeneration();
	    boolean isAlive(int row, int col);
	    void setCell(int row, int col, boolean alive);
	    int getRows();
	    int getCols();
	    ICell getCell(int x, int y);
	    IGrid getGrid();
	    void resetGrids();
	    String gridRep();
	}

Abbiamo quindi definito la struttura di queste entità. Ora definiamo il comportamento che ci aspettiamo da esse.

CellTest

	public class CellTest {
	    private ICell c;

	    @Test
	    public void testCellAlive() {
	        c.setStatus(true);
	        assertTrue(c.isAlive());
	    }

	    @Test
	    public void testCellDead() {
	        c.setStatus(false);
	        assertFalse(c.isAlive());
	    }
	}

GridTest

	public class GridTest {

	    private Grid grid;

	    @Before
	    public void setup() {
	        grid = new Grid(5, 5);
	    }

	    @Test
	    public void testDims() {
	        assertTrue(grid.getRowsNum() == 5);
	        assertTrue(grid.getColsNum() == 5);
	    }

	    @Test
	    public void testGridCellValue() {
	        grid.setCellValue(0, 0, true);

	        assertTrue(grid.getCellValue(0, 0));   // la cella impostata è viva
	        assertFalse(grid.getCellValue(0, 1));  // cella vicina non modificata = morta
	    }
	}

LifeTest

	public class LifeTest {

	    private LifeInterface life;

	    @Before
	    public void setup() {
	        life = new Life(5, 5);
	    }

	    @Test
	    public void testResetGrids() {
	        life.setCell(1, 1, true);
	        life.setCell(3, 4, true);

	        life.resetGrids();

	        for (int r = 0; r < life.getRows(); r++) {
	            for (int c = 0; c < life.getCols(); c++) {
	                assertFalse(life.isAlive(r, c));
	            }
	        }
	    }

	    @Test
	    public void testBlinkerOscillation() {
	        life.resetGrids();
	        life.setCell(1, 2, true);
	        life.setCell(2, 2, true);
	        life.setCell(3, 2, true);

	        life.nextGeneration();

	        assertTrue(life.isAlive(2, 1));
	        assertTrue(life.isAlive(2, 2));
	        assertTrue(life.isAlive(2, 3));
	    }

	    @Test
	    public void testLonelyCellDies() {
	        life.resetGrids();
	        life.setCell(2, 2, true);

	        life.nextGeneration();

	        assertFalse(life.isAlive(2, 2));
	    }
	}
